# SnapLocal

一个类似 SnapAny 的本地部署版视频 / 图片 / 音频链接解析下载工具。

## 启动

```bash
node server.js
```

打开浏览器访问：

```text
http://localhost:3000
```

## 当前能力

- 支持粘贴链接并识别常见平台来源。
- 支持直接媒体文件链接下载，例如 `.mp4`、`.webm`、`.jpg`、`.png`、`.mp3`。
- 内置最近解析记录、深色模式、移动端适配、FAQ。
- 后端解析入口集中在 `server.js` 的 `/api/parse`，方便替换成真实解析服务。

## 说明

社交平台链接的真实媒体下载通常需要平台允许的 API、授权账号或合规解析服务。当前项目提供本地可运行的界面、识别逻辑和扩展接口，不会绕过第三方平台限制。
